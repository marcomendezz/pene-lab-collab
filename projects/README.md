# Projects — Pene Lab

Shared project workspace for Marco, Gonzalo, Hermes, and Jarvis.

## Structure
```
projects/
├── [project-name]/    ← Each project gets its own folder
│   ├── README.md      ← Project brief, stack, requirements
│   ├── src/           ← Source code
│   ├── docs/          ← Project docs
│   └── .env.example   ← Environment template (never commit real .env)
```

## Workflow
1. Marco/Gonzalo create project brief in README.md
2. Bots pick up tasks and code in src/
3. All changes go through git (commit + push)
4. Use VS Code Live Share for real-time human collab
5. Tag bots in Telegram for code reviews or new features
